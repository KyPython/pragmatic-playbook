# Apollo Cold Email Campaign - Quick Start

Get your first 30-50 emails sent this week in 1 hour.

## 1. Set Up Apollo (30 minutes)

### Create Account
1. Go to [apollo.io](https://www.apollo.io) → Sign Up
2. Choose plan (Free or Basic $49/month)
3. Verify email

### Configure Workspace
1. Settings → Workspace → Name: "Founders Infrastructure"
2. Settings → Email → Add domain: `foundersinfra.com`
3. Add DNS records (SPF, DKIM, DMARC)
4. Settings → Email → Add sender: `founders@foundersinfra.com`
5. Verify domain (5-10 minutes)

## 2. Create ICP List (15 minutes)

1. Lists → Create List
2. Name: "Seed-Series A CTOs - Velocity Wall"
3. Apply filters:
   - Title: CTO, VP Engineering, Head of Engineering
   - Company Size: 11-50
   - Industry: Software, Internet
   - Funding: Seed OR Series A
   - Location: United States
   - Technology: Node.js OR Python OR AWS
4. Save (should have 300-800 contacts)

## 3. Set Up Sequence (15 minutes)

1. Sequences → Create Sequence
2. Name: "Consulting Outreach"
3. Copy emails from `07-apollo-import-ready-sequences.md`
4. Add 3 emails (Day 0, 3, 7)
5. Set automation rules
6. Test with your email

## 4. Launch First Batch (5 minutes)

1. Select first 30-50 contacts from list
2. Add to sequence
3. Verify sender: founders@foundersinfra.com
4. Start sequence

## 5. Track Results (Daily)

Use `05-email-tracking-template.md` to track:
- Opens
- Replies
- Qualified leads
- Booked calls

## Expected Results (First 50 Emails)

- Opens: 15-18 (30-35%)
- Replies: 1-2 (3-5%)
- Qualified: 1
- Booked calls: 0-1
