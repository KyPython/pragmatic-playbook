# Apollo Campaigns for Consulting Services

Complete setup for Apollo cold outreach targeting **Seed/Series A CTOs** for consulting services ($1,500-$3,500).

## Target ICP

**Primary:** Seed/Series A CTOs with 5-15 engineers  
**Pain Point:** "We ship fast but now we're slow" (velocity wall)  
**Budget:** $1,500-$3,500 per engagement  
**Geography:** United States (primary)

## Quick Start

1. **Review ICP Profile** → `01-consulting-icp-profile.md`
2. **Set Up Apollo** → Follow `03-apollo-setup-checklist.md`
3. **Use Email Sequences** → Copy from `02-email-sequences.md`
4. **Launch Campaign** → Start with 50-100 contacts

## Files

- **01-consulting-icp-profile.md** - Ideal customer profile, filters, messaging
- **02-email-sequences.md** - Email templates for campaigns
- **03-apollo-setup-checklist.md** - Step-by-step setup guide

## Expected Results

- **Open Rate:** 30-35%
- **Reply Rate:** 3-5%
- **Booked Calls:** 1% of emails sent
- **Closed Deals:** 0.2% of emails sent (1-2 per 1,000 contacts)
