# Email Templates - Styled HTML

These are production-ready HTML email templates that match your landing page branding (blue theme).

## Templates Available

1. **01-welcome-email.html** - Welcome email for new subscribers
2. More templates coming...

## Brand Colors

- **Primary Blue**: `#3b82f6` (blue-500)
- **Dark Blue**: `#1e40af` (blue-800)
- **Gradient**: `linear-gradient(135deg, #3b82f6 0%, #1e40af 100%)`
- **Text**: `#333` (dark gray)
- **Background**: `#f8f9fa` (light gray)

## Using in HubSpot

1. **Copy HTML:**
   - Open the template file
   - Copy all HTML content

2. **Create Email in HubSpot:**
   - Go to Marketing â†’ Email
   - Create new email
   - Switch to HTML editor
   - Paste template HTML

3. **Add Personalization:**
   - Replace `{{first_name}}` with HubSpot personalization: `{{contact.firstname}}`
   - Replace `{{your_name}}` with your actual name
   - Or use HubSpot's personalization tokens

4. **Test:**
   - Send test email to yourself
   - Check on mobile and desktop
   - Verify links work

## Template Variables

- `{{first_name}}` - Contact's first name
- `{{your_name}}` - Your name (replace manually)
- `{{company.name}}` - Company name (if available)

## Mobile Responsive

All templates are mobile-responsive and tested on:
- Desktop (600px width)
- Mobile (320px+ width)
- Email clients (Gmail, Outlook, Apple Mail)
